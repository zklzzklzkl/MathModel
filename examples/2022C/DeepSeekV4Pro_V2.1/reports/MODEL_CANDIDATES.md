## Route A: 统计保守路线 (Conservative Statistical Baseline)

**适用问题类型**: 全4问
**路线哲学**: 经典统计 + 线性方法, 可解释性强，实现风险低

### Q1a — 风化与类型/纹饰/颜色关系

- **方法**: 卡方独立性检验 + 对应分析(CA)
- **假设**: 样本量足够(风化=34, 无风化=24)，频数>5的单元格≥80%
- **变量**: 风化(binary), 类型(categorical 2类), 纹饰(3类), 颜色(8类含缺失)
- **算法**: 
  1. 构建风化×类型、风化×纹饰、风化×颜色列联表
  2. χ²检验，计算Cramér's V效应量
  3. 对应分析降维可视化
- **输出**: 列联表 + p值表 + Cramér's V表 + 对应分析双标图

### Q1b — 分类型风化前后成分统计规律

- **方法**: 分组描述统计 + 独立样本t检验(或Mann-Whitney U)
- **假设**: 各组分近似正态(或使用非参数替代)
- **变量**: 12种有效组分(排除SnO2/SO2)
- **算法**:
  1. 数据归一化至100%
  2. 按 类型(高钾/铅钡) × 风化(是/否) 分4组
  3. 计算每组均值±标准差, 变异系数
  4. 组间差异显著性检验(p < 0.05)
- **输出**: 分组统计表 + 箱线图 + 雷达图 + 显著性热图

### Q1c — 风化前成分预测

- **方法**: 配对差值比例模型
- **假设**: 同类型玻璃的风化变化模式一致
- **算法**:
  1. 从3对配对数据(08/26/54)计算每个组分的风化变化率: r_j = (X_pre - X_post) / X_pre
  2. 对铅钡玻璃采用平均变化率反演: X̂_pre = X_post / (1 - r̄_j)
  3. 对高钾玻璃利用未风化点统计作为参照
  4. 验证: 留一交叉验证(3对)
- **输出**: 各组分变化率表 + 预测值 vs 实测值对比图

### Q2a — 分类规律

- **方法**: 组分阈值规则 + 线性判别分析(LDA)
- **关键区分组分**: PbO, BaO, K2O, SiO2
- **算法**:
  1. 基于已知标签建立阈值规则: K2O>3%且PbO<5%→高钾; PbO>8%→铅钡
  2. LDA交叉验证准确率
  3. 特征重要性排序
- **输出**: 分类规则表 + LDA投影散点图 + 混淆矩阵

### Q2b — 亚类划分

- **方法**: K-means聚类 + 轮廓系数选择k
- **假设**: 亚类在成分空间中呈球形分布
- **算法**:
  1. PCA降维至2-3维
  2. 对高钾(18个)和铅钡(40个)分别K-means, k=2,3,4
  3. 轮廓系数选择最优k
  4. 对各亚类进行统计描述
- **输出**: 聚类结果表 + PCA散点图(按亚类着色) + 轮廓系数图

### Q3 — 未知样品分类

- **方法**: 马氏距离判别
- **算法**:
  1. 计算每类(高钾/铅钡)的均值向量μ和协方差矩阵Σ
  2. 对每个未知样品计算到两类的马氏距离
  3. 归入距离较小的一类
  4. 敏感性: ±5%组分扰动, 重新计算类别
- **输出**: 分类结果表 + 马氏距离图 + 敏感性分析表

### Q4 — 成分关联关系

- **方法**: Pearson相关系数矩阵 + Fisher Z检验比较组间差异
- **算法**:
  1. 对高钾和铅钡分别计算12组分相关系数矩阵
  2. Fisher Z transformation比较两组相关系数差异显著性
  3. 识别显著差异的组分对
- **输出**: 两组热力图 + 差异显著性矩阵 + 关键组分对散点图

### Route A 优势
- 方法成熟，可解释性强
- 代码实现简单，出错概率低
- 所有步骤可追溯

### Route A 风险
- Q1c仅依赖3对数据，预测精度有限
- 简单阈值规则可能不够优雅
- 亚类划分较粗糙

## Route B: 高分增强路线 (High-Score Enhanced)

**路线哲学**: 贝叶斯 + 鲁棒性 + 多维验证，追求高分论文标准

### Q1a — 风化与类型/纹饰/颜色关系

- **方法**: 多重对应分析(MCA) + Fisher精确检验 + 逻辑回归
- **创新点**: 以风化为因变量建立logistic回归，量化各因素对风化概率的影响
- **变量**: 类型, 纹饰, 颜色 → 风化概率
- **算法**:
  1. MCA降维可视化多维分类变量
  2. 单因素Fisher精确检验
  3. 多因素logistic回归: logit(P_风化) = β₀ + β₁×类型 + β₂×纹饰 + β₃×颜色
  4. 计算优势比(OR)和95%CI
- **输出**: MCA双标图 + 优势比森林图 + logistic回归表 + ROC曲线

### Q1b — 风化前后成分统计规律

- **方法**: 多变量方差分析(MANOVA) + 效应量(Hedges' g) + Bootstrap置信区间
- **创新点**: Bootstrap重采样(1000次)获得均值的稳健置信区间
- **算法**:
  1. 数据归一化
  2. MANOVA检验风化×类型的交互效应
  3. 对各组分计算Hedges' g效应量(风化vs非风化)
  4. Bootstrap CI
- **输出**: 效应量排序图 + Bootstrap箱线图 + 交互效应表

### Q1c — 风化前成分预测

- **方法**: 贝叶斯层次模型
- **假设**: 风化变化率在不同样品间有所变化但服从同一先验分布
- **算法**:
  1. 利用3对直接配对+10对未风化点配对建立层次模型:
     y_post = y_pre × (1 - δ) + ε, δ ~ Beta(a,b)
  2. MCMC采样估计后验分布
  3. 预测: 从后验采样得到风化前的后验均值和95%可信区间
  4. 对比方法: 同时实现路线A的简单比例模型作为对比
- **输出**: 后验分布图 + 预测值含区间 + 方法对比表

### Q2a — 分类规律

- **方法**: 组分比值特征工程 + 正则化Logistic回归 + Fisher判别验证
- **创新点**: 构造有物理意义的比值特征
- **算法**:
  1. 构造比值特征: PbO/SiO2, BaO/K2O, (PbO+BaO)/SiO2, K2O/SiO2, Al2O3/CaO
  2. 可视化高钾/铅钡在这些比值上的分离度
  3. 建立L1正则化logistic回归(稀疏特征选择)
  4. 交叉验证准确率
- **输出**: 比值散点矩阵 + 正则化路径图 + 分类决策边界图 + CV准确率

### Q2b — 亚类划分

- **方法**: 层次聚类(Ward) + Gap统计量 + 轮廓系数联合优化
- **创新点**: 多种聚类方法对比 + 稳定性分析
- **算法**:
  1. 对高钾和铅钡分别进行标准化的层次聚类(Ward linkage)
  2. Gap统计量确定最优k
  3. 对比K-means和GMM的结果一致性(ARI指标)
  4. 稳定性分析: 对每个亚类逐个剔除样本，检查聚类标签变化率
- **输出**: 谱系图 + Gap曲线 + ARI一致性热图 + 亚类统计雷达图

### Q3 — 未知样品分类

- **方法**: 多分类器集成(马氏距离 + LDA + QDA + KNN) + 加权投票
- **创新点**: 多分类器一致性评估 + 全面的敏感性分析
- **算法**:
  1. 4种分类器独立预测, 记录概率/距离
  2. 加权投票确定最终类别(权重=各分类器CV准确率)
  3. 一致性分析: Fleiss' κ
  4. 敏感性分析: 
     a. 组分±5%扰动 → 分类稳定性
     b. 留一训练集扰动 → 分类一致性
     c. 风化样品成分矫正后再分类
- **输出**: 多分类器结果矩阵 + 分类概率条形图 + 扰动敏感性热图 + 风化矫正前后对比

### Q4 — 成分关联关系

- **方法**: 偏相关网络 + Graphical Lasso + 差异网络分析
- **创新点**: 图模型揭示条件独立结构
- **算法**:
  1. 分离钾和铅钡两组, 计算Pearson和Spearman相关矩阵
  2. Graphical Lasso估计稀疏偏相关网络(正则化参数λ由BIC选择)
  3. Fisher Z检验比较两组每个组分对的相关系数差异
  4. 构造差异网络, 标识显著不同的关联边
  5. 社区检测(Modularity)揭示组分模块
- **输出**: 相关网络对比图(高钾 vs 铅钡) + 差异网络图 + 社区结构表 + 差异显著性表

### Route B 优势
- 每问都有创新方法和对比基准
- 多层敏感性/稳定性分析
- 统计推断严谨(Bayesian CI, bootstrap, permutation test)
- 论文更有深度, 更容易得高分

### Route B 风险
- 代码实现复杂, 部分方法(Bayesian MCMC)调试成本高
- Graphical Lasso在样本量较小时(高钾18个)可能不稳定
- 时间成本较高
