## 2. Background Summary

### 2.1 Domain Context

The problem is set at the intersection of archaeology and analytical chemistry, specifically concerning ancient glass artifacts from the Silk Road era.

**Historical setting:**
- Glass originated in Western Asia and Egypt as bead ornaments, then traveled east along the Silk Road into China.
- After absorbing foreign techniques, ancient Chinese artisans began locally sourcing raw materials, producing visually similar but chemically distinct glass.
- Two primary glass types define the dataset:
  - **高钾玻璃 (High-Potassium Glass):** Fluxed with plant ash (草木灰), rich in K2O. Prevalent in Lingnan (southern China), Southeast Asia, and India.
  - **铅钡玻璃 (Lead-Barium Glass):** Fluxed with lead ore (铅矿石), rich in PbO and BaO. Believed to be a uniquely Chinese invention, associated with Chu culture.

**Chemical basis:**
- Quartz sand (SiO2) is the primary raw material but has a high melting point (~1700 C).
- Fluxes (助熔剂) lower the melting temperature: plant ash, natural natron (泡碱), saltpeter (硝石), lead ore.
- Limestone (石灰石) acts as a stabilizer, converting to CaO upon calcination.
- Different flux choices produce measurably different chemical profiles.

**Weathering problem:**
- Buried glass undergoes weathering, exchanging internal elements with the burial environment.
- This alters composition ratios, potentially misleading classification.
- The dataset distinguishes: unweathered surfaces, weathered surfaces, unweathered points on weathered artifacts, and severely weathered points from the weathering layer.

### 2.2 Available Data

**Form 1** (表单1): Basic artifact information -- 58 rows (文物编号 01-58), 5 columns:
- 文物编号 (Artifact ID)
- 纹饰 (Decoration pattern): A, B, or C

- Same chemical components + a 表面风化 (weathering) column

## 3. Question-by-Question Deconstruction

### 3.1 Question 1: Weathering Analysis and Pre-Weathering Prediction

**原文:** 对这些玻璃文物的表面风化与其玻璃类型、纹饰和颜色的关系进行分析；结合玻璃的类型，分析文物样品表面有无风化化学成分含量的统计规律，并根据风化点检测数据，预测其风化前的化学成分含量。

**Mathematical task types:**
- **Part A (Relationship analysis):** Categorical association / contingency analysis between weathering status and glass type, decoration, color. Chi-square tests, Cramer's V effect size, or correspondence analysis.
- **Part B (Statistical regularity):** Comparative statistical analysis of chemical composition distributions for weathered vs. unweathered samples, stratified by glass type. Hypothesis testing (t-test, Mann-Whitney U), effect size estimation.
- **Part C (Prediction):** Inverse prediction / calibration problem. Given weathered composition, predict pre-weathering composition. This is a regression or conversion-model problem.

**Inputs needed:**
- Form 1: type, decoration, color, weathering status for each artifact
- Form 2: chemical composition for all sampling points (both weathered and unweathered)
- Paired samples (same artifact, weathered + unweathered readings) are especially valuable

**Outputs expected:**
- Statistical evidence for/against relationships between weathering and categorical attributes
- Quantitative characterization of which chemical elements increase/decrease upon weathering, by glass type
- Predicted pre-weathering composition for weathered sampling points

**Modeling approaches:**
- For Part A: logistic regression (weathering ~ type + 纹饰 + 颜色) or multinomial association measures
- For Part B: per-element comparison of means/medians between weathered and unweathered within each glass type; compositional data approaches (CLR/ILR transform before comparison)
- For Part C: linear regression models per element (weathered -> original), built on paired-sample data. Alternatively, a stoichiometric correction model assuming certain elements are weathering-stable references. Or a ratio-preserving model using immobile elements as internal standards.

### 3.2 Question 2: Classification Refinement and Sub-classification

**原文:** 依据附件数据分析高钾玻璃、铅钡玻璃的分类规律；对于每个类别选择合适的化学成分对其进行亚类划分，给出具体的划分方法及划分结果，并对分类结果的合理性和敏感性进行分析。

**Mathematical task types:**
- **Part A (Classification rule discovery):** Identifying which chemical elements best discriminate between high-potassium and lead-barium glass. Feature importance / variable selection problem.
- **Part B (Sub-classification):** Within each primary type, conduct cluster analysis to discover sub-types and define classification rules. Unsupervised learning.
- **Part C (Validation & sensitivity):** Assess robustness of sub-classification to data perturbation, choice of clustering method, and variable selection.

**Inputs needed:**
- Form 2: chemical compositions for all classified samples
- Form 1: glass type labels

**Outputs expected:**
- Key discriminant elements for high-K vs. Pb-Ba separation
- Clustering results: number of sub-classes per glass type, membership assignments
- Explicit sub-classification rules (e.g., threshold-based decision trees)
- Sensitivity analysis: stability under cross-validation, perturbation, or alternative methods

**Modeling approaches:**
- Feature selection: t-test/Mann-Whitney per element between glass types; LDA/PLS-DA loading analysis; random forest feature importance
- Sub-classification: hierarchical clustering (Ward's method with Euclidean/Manhattan distance on scaled data), k-means (with elbow/silhouette for k selection), or Gaussian mixture models
- Rule definition: decision trees or simplified threshold rules derived from cluster centroids
- Sensitivity: bootstrap resampling to check cluster stability; compare multiple clustering algorithms (k-means vs. hierarchical vs. DBSCAN); measure adjusted Rand index consistency

## 5. Explicit Constraints

1. **Composition validity range:** Chemical composition sums between 85%-105% are treated as valid data. Samples outside this range must be flagged (and possibly excluded or analyzed separately).

2. **Compositional nature:** All variables are proportions of a whole. The sum should ideally be 100%. This is a compositional data (CoDa) structure, requiring special statistical treatment:
   - Standard Euclidean geometry is inappropriate (spurious correlations from closure)
   - CLR (centered log-ratio), ILR (isometric log-ratio), or ALR (additive log-ratio) transformations should be applied before many statistical operations
   - Correlation analysis (Q4) is particularly affected

3. **Missing = undetected:** Blank cells in Form 2 and Form 3 indicate the component was not detected. This represents left-censored data (below detection limit), not truly missing data. Treatment options: zero imputation, detection-limit imputation, or treating as structural zeros.

4. **Multi-site sampling:** Some artifacts have multiple sampling points (部位1, 部位2, 未风化点, 严重风化点). The structure of sampling must be preserved -- 部位1 and 部位2 are different locations on the same artifact and may differ chemically.

5. **Paired weathered/unweathered data:** Some artifacts have both weathered and unweathered measurements (e.g., artifacts 08, 26 have 严重风化点 paired data; artifacts 23, 25, 28, 29, 42, 44, 49, 50, 53 have 未风化点 data). These pairs are the key calibration data for Q1 Part C.

## 7. Ambiguities and Gaps

### 7.1 Data-Level Ambiguities

1. **Treatment of sub-100% sums:** When the 14-component sum is below 85%, is this a data quality issue or evidence of unmeasured components (e.g., H2O, CO2 from carbonates, organic matter)? The problem statement acknowledges detection limitations but does not specify what to do with out-of-range samples.

2. **Undetected (=blank) components:** No explicit detection limit is given for any element. The appropriate imputation method (zero vs. DL/2 vs. more sophisticated left-censored approaches) must be decided and justified.

3. **Color missingness:** Several artifacts (19, 40, 48, 58) lack color data. How should these be handled in the Q1 relationship analysis? Imputation is not straightforward for a nominal variable.

4. **"未风化点" (unweathered points on weathered artifacts):** Should these be treated as equivalent to samples from entirely unweathered artifacts? The problem states they are "unweathered regions on the surface of weathered artifacts" -- they may still be chemically affected by the burial environment.

5. **Sampling depth:** The chemical analysis method (likely XRF or similar surface technique for archaeological glass) is not specified, but typical surface analysis penetrates only microns. The representativeness of surface readings for bulk composition is an unstated assumption.

### 7.2 Modeling Ambiguities

6. **Q1 Part C -- "predict pre-weathering composition":** It is ambiguous whether the model should:
   - Predict all 14 components simultaneously (multivariate output), or
   - Model each component independently, or
   - Use a physical model (e.g., immobile-element normalization)
   
   Each approach has different mathematical rigor and complexity.

7. **Q2 -- "appropriate chemical components for sub-classification":** "Appropriate" is undefined. Criteria could include: statistical discrimination power, archaeological interpretability, or parsimony. The team must define and justify their criteria.

8. **Q2 -- "number of sub-classes":** No guidance on how many sub-classes are expected. Purely data-driven cluster number selection (elbow, silhouette, gap statistic) must be used and cross-validated.

9. **Q3 -- "sensitivity analysis":** The scope of sensitivity is unspecified. Could include: input perturbation, classifier choice, feature subset choice, or sample removal. Scope must be bounded.

10. **Q4 -- "association relationships":** The term is broad. Could mean linear correlation, monotonic association, partial correlation, mutual information, or causal/production relationships. The compositional nature of the data demands special treatment.

### 7.3 Contextual Ambiguities

11. **Glass-making recipes vs. measured composition:** The measured composition reflects both the original recipe AND post-burial alteration. Are we modeling the original production technology or the archaeological reality?

12. **Geographic/temporal information:** No provenance data (excavation site, estimated date) is provided, though the problem mentions Chu culture and Lingnan regions. This limits archaeological interpretation.

## 8. Recommended Modeling Directions

### 8.1 Question 1

**Part A (Weathering ~ categorical factors):**
- Cross-tabulation with chi-square tests
- Logistic regression: logit(P(风化)) ~ 类型 + 纹饰 + 颜色
- Cramer's V for effect size quantification
- Visual: mosaic plots, correspondence analysis biplot

**Part B (Statistical regularity of composition change):**
- Per-element comparison: weathered mean vs. unweathered mean within each glass type
- Mann-Whitney U test (nonparametric, handles non-normal compositional data)
- CLR transform before comparing means; report log-ratio differences
- Identify elements that consistently increase (enrichment from environment: e.g., P2O5, SO2, some oxides) vs. decrease (leaching: Na2O, K2O) upon weathering
- Use immobile elements (Al2O3, Fe2O3, possibly TiO2 if present) as internal references

**Part C (Pre-weathering prediction):**
- **Approach 1 (Immobile-element normalization):** Assume Al2O3 and/or Fe2O3 are immobile. Normalize weathered composition by the enrichment factor of immobile elements relative to unweathered reference.
- **Approach 2 (Paired regression):** Use artifacts with both weathered and unweathered data (08, 23, 25, 26, 28, 29, 42, 44, 49, 50, 53) to build per-element linear models: `C_original = a + b * C_weathered`. Apply to Q3 weathered unknowns.
- **Approach 3 (Ratio preservation):** Assume ratios of certain element pairs are preserved through weathering. Model the systematic bias.
- **Approach 4 (Mass balance model):** Assume weathering adds/subtracts mass. The sum change from 100% quantifies net gain/loss. Distribute changes proportionally.

Recommend combining Approach 1 (for physical basis) and Approach 2 (for data-driven calibration), then comparing results.

### 8.2 Question 2

**Step 1 -- Discriminant element identification:**
- Per-element t-test between 高钾 and 铅钡 groups (on CLR-transformed data)
- LDA with stepwise variable selection
- Random forest feature importance ranking
- Expected key discriminants: PbO, BaO (high in 铅钡), K2O (high in 高钾)

**Step 2 -- Sub-classification within each type:**
- CLR-transform data first (critical for compositional data)
- Standardize (z-score) after CLR
- Apply hierarchical clustering (Ward's method, Euclidean distance)
- Determine k via: elbow plot, silhouette score, gap statistic, and dendrogram inspection
- Additionally try: k-means, GMM (Gaussian Mixture Model) for comparison
- Define interpretable rules: decision tree on original-scale key elements

**Step 3 -- Validation and sensitivity:**
- Internal validation: silhouette scores, Dunn index
- Stability: bootstrap resampling -> cluster consensus (adjusted Rand index)
- Comparison: compute ARI between different clustering methods
- Interpret sub-classes archaeologically (do they correspond to known glass recipes?)

### 8.3 Question 3

**Step 1 -- Data preparation:**
- For weathered unknowns (A2, A5, A6, A7): apply Q1 Part C weathering correction model
- Verify corrected sums fall within 85%-105%

**Step 2 -- Classification:**
- Apply Q2 decision rules for 高钾/铅钡 separation
- Use multiple classifiers: LDA (with CLR or key-element inputs), k-NN (k=3,5), Random Forest, SVM

**Step 3 -- Sub-class assignment:**

### 8.4 Question 4

**Step 1 -- Data preparation:**
- Use Q1-corrected compositions for all samples
- Group by: Q2 primary type, Q2 sub-class
- CLR-transform all data first (essential for valid correlation on compositional data)

**Step 2 -- Correlation structure analysis:**
- Compute CLR-based variation matrix (log-ratio variances between all element pairs)

- Partial correlation networks (graphical lasso) to reveal conditional dependencies
- Identify element clusters that co-vary (possible shared mineral sources or flux recipes)

**Step 3 -- Between-group comparison:**
- Permutation test: shuffle group labels, recompute correlation difference, build null distribution
- Compare partial correlation network edge sets between types
- Procrustes analysis on group-specific PCA loadings
- Interpret differences in terms of glass-making technology: different flux minerals -> different element co-occurrence patterns

**Step 4 -- Archaeological interpretation:**
- High-K glass: expect K2O correlations with MgO, P2O5 (all from plant ash)
- Pb-Ba glass: expect PbO-BaO co-variation (shared lead ore source); SiO2 inverse correlation with PbO+BaO
- Sub-class level differences may reveal distinct workshop traditions or raw material sources

## 9. Summary of Key Considerations

1. **Compositional data is the defining technical challenge.** CLR/ILR transforms are non-negotiable for valid correlation, PCA, and distance-based clustering.

2. **Q1 and Q2 are load-bearing.** Every subsequent question depends on weathering correction and classification rules. These must be solved with rigorous validation before proceeding.

3. **Paired samples are golden.** The 11+ artifacts with both weathered and unweathered measurements are the calibration backbone for weathering correction. Preserve and exploit these pairs.

4. **Missing data decisions must be explicit.** The team must document and justify all choices: zero imputation vs. DL imputation for undetected elements, handling of out-of-range sums, and treatment of missing color values.

5. **Sensitivity analysis appears in Q2 and Q3.** Both require formal perturbation analysis, not just qualitative discussion. Bootstrap resampling is the recommended general approach.

6. **Model interpretability matters.** This is an archaeological problem -- classification rules should be explainable in terms of glass chemistry (which flux materials, which colorants), not just "the random forest voted yes."
