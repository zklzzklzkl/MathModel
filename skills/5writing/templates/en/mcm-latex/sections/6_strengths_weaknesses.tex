\section{Strengths and Weaknesses}

\subsection{Strengths}
\begin{itemize}
  \item Interpretable model structure with clear mathematical foundations.
  \item Ensemble approach provides robustness against overfitting.
  \item Sensitivity analysis confirms model stability across parameter ranges.
  \item Results are reproducible with provided code and data.
\end{itemize}

\subsection{Weaknesses}
\begin{itemize}
  \item Assumes data quality and absence of systematic measurement errors.
  \item Computational cost grows with ensemble size.
  \item May not fully capture non-linear interaction effects without further tuning.
\end{itemize}
