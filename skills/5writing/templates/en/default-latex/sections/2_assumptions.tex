\section{Assumptions and Justifications}

To simplify the real-world problem, we make the following assumptions:

\begin{enumerate}
  \setlength{\itemsep}{0.9em}
  \item The provided data is accurate and free from systematic measurement errors.
  \item Interaction effects between different factors are negligible.
  \item The system remains in a steady state over the short term.
  \item All variables are continuous and differentiable.
  \item Missing values can be handled through imputation techniques.
\end{enumerate}
