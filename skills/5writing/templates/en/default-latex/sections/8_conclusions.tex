\section{Conclusions}

Our main contributions:

\begin{enumerate}
  \setlength{\itemsep}{0.9em}
  \item Developed a robust prediction framework achieving $R^2 = 0.923$.
  \item Implemented effective feature selection for identifying key factors.
  \item Designed an optimization algorithm for constrained problems.
  \item Validated robustness through comprehensive sensitivity analysis.
\end{enumerate}

The framework achieves high accuracy while maintaining interpretability,
and can be extended to various real-world applications.
